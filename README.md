# html-cv
CV Html Markup
README — My CV Project

This project contains my personal CV designed using HTML and CSS.
The file can be opened in any browser and exported as a PDF.

How to Use

1. Open the file in a text editor.


2. Edit your name, contact details, education, skills, and experience.


3. Save and open it in a browser to preview.


4. To download as a PDF: open the browser’s print menu → select Save as PDF.
